This script creates interesting visual effects using the Windows GDI (Graphics Device Interface). It's a demonstration of how Python can interact with the Windows display system. 

import win32gui
import win32con
import ctypes
import math
import random

user32 = ctypes.windll.user32
user32.SetProcessDPIAware()
[sw, sh] = [user32.GetSystemMetrics(0), user32.GetSystemMetrics(1)]
hdc = win32gui.GetDC(0)

while True:

    win32gui.BitBlt(
        hdc,
        0,
        0,
        sw,
        sh,
        hdc,
        random.randrange(-3, 4),
        random.randrange(-3, 4),
        win32con.NOTSRCCOPY,
    )
Note: This script creates a subtle screen distortion effect and requires the pywin32 library to be installed. Make sure you're running Windows before using it.
